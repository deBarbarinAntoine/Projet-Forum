-- Liste des threads d'une catégorie -- 

    -- Valeur -- 

  - Threads appartenant à la catégorie en question 
  - Titre du thread / Auteur du thread / Date de création du thread 
  - Savoir si le thread est Fermé / Ouvert 
  
    -- Fonction -- 

  - Fonction de recherche dans la liste des threads
  - Possibilité de mettre en favoris un thread
  - Possibilité de créer un thread 
  - Si admin : possibilité de supprimer le thread 



-- Post dans un thread -- 
    
    -- Valeur -- 

 - Tous les posts du thread 
 - Auteur du post / Date du post 
 - Nombre de likes du post (emoji)
 
    -- Fonction -- 

 - Fonction de recherche dans le thread 
 - Possibilité de mettre en fav un post
 - Possibilité de liker un post
 - Réponse à un post précis
 - Écrire une réponse à la suite des autres posts
 - Pagination 



 -- Création d'un thread -- 
    
 - Titre du thread 
 - Message du thread (équivalent au 1er post)
 - Bouton de création du thread



 -- Page de recherche -- 

    -- Valeur --

 - Tags par rapport à la recherche
 - Utilisateur par rapport à la recherche
